# AI-Based Cloud Cost Optimization System

## 📌 Project Description
This project helps reduce AWS cloud costs by automatically monitoring and stopping unused EC2 instances.

## 🚀 Features
- Auto shutdown of idle EC2 instances
- Cost optimization using automation
- AWS resource monitoring
- Beginner-friendly cloud project

## 🛠️ Technologies Used
- AWS EC2
- Python (boto3)
- Git & GitHub

## ⚙️ How It Works
1. Connects to AWS using boto3
2. Checks all EC2 instances
3. Detects running instances
4. Stops unused instances automatically
